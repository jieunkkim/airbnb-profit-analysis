# 프로젝트 가이드

## 버전 관리 (Semantic Versioning)

버전을 올릴 때마다 package.json의 version을 자동으로 수정하세요:

- **PATCH (v1.0.X)**: 버그 수정, UI 미세 조정, 텍스트 수정 → 예: v1.0.1
- **MINOR (v1.X.0)**: 새 기능 추가, 기존 기능 개선 → 예: v1.1.0
- **MAJOR (vX.0.0)**: 구조 변경, 하위호환 불가능한 변경 → 예: v2.0.0

커밋 메시지가 `feat:` 또는 `fix:` 로 시작하면, 해당 변경사항에 맞게 자동으로 버전을 올리고 커밋하세요.
